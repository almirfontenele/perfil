# Almir Fontenele - Landing Page de Perfil Profissional

## Visão Geral

Landing page moderna e responsiva para apresentação do perfil profissional de **Almir Fontenele**, Engenheiro de Software Sênior especializado em Java, AWS e integração com IA/LLM.

## Funcionalidades Implementadas

### Design e UX
- **Tema escuro moderno** com gradientes em azul/ciano
- **Layout totalmente responsivo** (desktop, tablet, mobile)
- **Animações suaves** com Intersection Observer
- **Efeito de partículas** no hero section
- **Menu de navegação fixo** com efeito de scroll
- **Indicador de scroll** animado

### Seções da Página
1. **Hero Section** - Apresentação principal com stats e CTAs
2. **Sobre Mim** - Trajetória e expertise em cards
3. **Habilidades** - Top skills com barras de progresso animadas
4. **Certificações** - Timeline visual das formações
5. **Patente** - Destaque para o SC-MIDWARECONNECTOR
6. **Idiomas** - Cards de idiomas (PT/EN)
7. **Contato** - Formulário + informações de contato

### Interatividade
- Navegação suave entre seções
- Animação de barras de skills ao scroll
- Botão "voltar ao topo"
- Menu mobile hambúrguer com animação
- Efeito parallax no hero
- Notificação toast no envio do formulário
- Suporte a `prefers-reduced-motion`

## Estrutura de Arquivos
